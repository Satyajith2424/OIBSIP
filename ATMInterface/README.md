# ATM Interface (TASK - 3)

## Objective

This project is a console-based ATM application developed using Java as part of the Oasis Infobyte Java Development Internship.

## Features

* User Login using User ID and PIN
* Deposit Money
* Withdraw Money
* Transfer Money
* View Transaction History
* Quit Application

## Technologies Used

* Java
* Eclipse IDE
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## Default Login Credentials

User ID: USER001

PIN: 1234

## Project Structure

* Main.java
* User.java
* Account.java
* Transaction.java
* ATMService.java

## How to Run

1. Open the project in Eclipse.
2. Run Main.java.
3. Enter the login credentials.
4. Use the menu options to perform ATM operations.
