# Online Examination System (TASK - 4)

## Objective

This project is a console-based Online Examination System developed using Java as part of the Oasis Infobyte Java Development Internship.

## Features

* User Login
* Update Password
* Start Online Examination
* Multiple Choice Questions (MCQs)
* Score Calculation
* Timer and Auto Submit
* Logout and Session Closing

## Technologies Used

* Java
* Eclipse IDE
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class
* Java Threads
