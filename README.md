# OIBSIP - Oasis Infobyte Java Development Internship

This repository contains the projects completed as part of the Oasis Infobyte Java Development Internship.

## Projects

### Task 3 - ATM Interface

A console-based ATM application developed using Java.

Features:

* Login
* Deposit
* Withdraw
* Transfer
* Transaction History
* Quit

### Task 4 - Online Examination System

A console-based online examination system developed using Java.

Features:

* Login
* Update Password
* MCQ Exam
* Timer and Auto Submit
* Score Calculation
* Logout

## Technologies Used

* Java
* Eclipse IDE
* Object-Oriented Programming (OOP)
* Java Collections
* Java Threads
